# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security issue in this portfolio project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Email **abhijithcsg8@gmail.com** with:
   - A description of the issue
   - Steps to reproduce (if applicable)
   - Any suggested fix (optional)

You can expect a response within **7 business days**. If the issue is confirmed, a fix will be prioritized and you will be notified when it is resolved.

## Scope

This policy applies to the source code in this repository only. Third-party dependencies are managed via npm; report upstream issues to their respective maintainers when appropriate.
